# 图片查看器 PRD

**版本**: v1.0 | **日期**: 2025-02-28

---

## 1. 功能分级

### P0 - MVP必备 (v0.1.0)
| ID | 功能 | 描述 |
|----|------|------|
| F-001 | 应用启动 | 支持命令行/双击启动，打开文件/目录 |
| F-002 | 图片打开 | Ctrl+O、支持 PNG/JPEG/GIF/WebP/BMP/TIFF |
| F-003 | 基础浏览 | 自适应窗口、1:1显示、上下张切换 (←/→) |
| F-004 | 缩放平移 | 滚轮缩放(10%-2000%)、拖拽平移 |
| F-005 | 全屏模式 | F11/双击切换，Esc退出 |
| F-006 | 中文界面 | 所有菜单、按钮、提示使用中文 |
| F-107 | 拖放打开图片 | 把图片文件拖到软件窗口直接打开查看 |
| F-109 | 关于窗口 | 菜单增加"关于"选项，显示版权声明和软件信息 |

### P1 - 重要功能 (v0.2.0)
| ID | 功能 | 描述 |
|----|------|------|
| F-101 | 缩略图画廊 | G键切换，异步加载，点击跳转 |
| F-102 | 快捷键系统 | 完整快捷键，?显示帮助 |
| F-103 | 配置系统 | TOML配置持久化 |
| F-104 | 信息面板 | F键切换，显示元数据 |
| F-105 | 显示原始尺寸 | 菜单/工具栏显示图片宽高 |
| F-106 | 右键复制图片 | 右键菜单复制到剪贴板 |
| F-108 | 右键打开方式 | 系统右键菜单集成 |

### P2 - 增强功能 (v1.0.0)
| ID | 功能 | 描述 |
|----|------|------|
| F-201 | 幻灯片模式 | S键启动，可配置间隔 |
| F-202 | EXIF显示 | 相机参数、GPS等信息 |
| F-203 | 基础编辑 | 旋转/翻转，另存为新文件 |
| F-204 | 主题系统 | 深/浅/纯黑主题 |
| F-205 | 图片旋转 | 顺时针/逆时针旋转90° |
| F-206 | 图片镜像翻转 | 水平/垂直翻转 |

## 2. 验收标准

### P0 验收 (v0.1.0)
- [x] 冷启动 ≤ 1秒
- [x] 支持 PNG/JPEG/GIF/WebP/BMP/TIFF
- [x] 滚轮缩放、Ctrl++/-/0
- [x] ←→ 切换图片
- [x] F11 全屏，Esc 退出
- [x] 所有界面文本为中文
- [x] 拖放图片文件到窗口可直接打开查看
- [ ] 菜单栏有"帮助"→"关于"，显示版本和版权信息

### P1 验收 (v0.2.0)
- [ ] G键切换画廊/单图
- [ ] 缩略图异步加载
- [ ] 配置持久化到平台路径
- [ ] F键显示信息面板
- [ ] 菜单/工具栏显示图片原始尺寸
- [ ] 右键菜单可复制图片到剪贴板
- [ ] 右键图片可选择使用本软件打开

### P2 验收 (v1.0.0)
- [ ] S键幻灯片，可配置3/5/10秒
- [ ] 显示EXIF相机参数
- [ ] R/Shift+R/H/V 旋转翻转快捷键
- [ ] 主题可配置
- [ ] 菜单/工具栏支持旋转翻转

### 性能指标
| 指标 | 目标 |
|------|------|
| 冷启动 | ≤ 1.0s |
| 打开4K图 | ≤ 1.0s |
| 切换图片 | ≤ 0.3s |
| 内存占用 | ≤ 500MB |

### 兼容性
- Windows 10+ (x64/ARM64)
- macOS 11+ (x64/ARM64)
- Linux Ubuntu 20.04+ (x64/ARM64)
